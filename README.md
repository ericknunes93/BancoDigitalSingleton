# Banco Digital - Singleton

Projeto desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos (POO) e a implementação do padrão de projeto Singleton.

## Tecnologias

- Java
- IntelliJ IDEA
- Git
- GitHub

## Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Classes e Objetos
- Métodos
- Singleton Design Pattern

## Estrutura do Projeto

```
src
├── Main.java
├── model
│   ├── Client.java
│   └── Account.java
└── singleton
    └── Banco.java
```

## Funcionalidades

- Implementação do padrão Singleton
- Cadastro de cliente
- Depósito em conta
- Saque com validação de saldo
- Consulta de saldo

## Como executar

1. Clone este repositório.
2. Abra o projeto em uma IDE Java (IntelliJ IDEA ou Eclipse).
3. Execute a classe `Main`.
4. Informe o nome do cliente, o valor do depósito e o valor do saque.
