# BancoDigitalSingleton

Projeto de Banco Digital implementando o padrão Singleton.

## Descrição

Este projeto demonstra a implementação do padrão de design Singleton em um sistema de banco digital.

## Como Usar

1. Clone o repositório
2. Instale as dependências
3. Execute o projeto

## Requisitos

- Node.js 16+
- npm ou yarn
